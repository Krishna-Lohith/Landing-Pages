## 🚀 Getting Started

**1. Clone the repository**

```bash
git clone https://github.com/Krishna-Lohith/Landing-Pages.git
cd Landing-Pages
```

**2. Open any page**

Navigate into a folder and open the `index.html` file directly in your browser — no setup needed!

```bash
# Example
open "Ibachi Steakhouse/index.html"
```
